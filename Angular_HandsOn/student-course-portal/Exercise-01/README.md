# Exercise 01 – Angular Project Setup & Component Generation

## Objective
The objective of this exercise is to understand the Angular project structure, create a new Angular application using Angular CLI, generate reusable components, configure routing, and build a simple Student Course Portal application.

---

## Tools Used
- Angular 20
- Node.js
- Angular CLI
- Visual Studio Code

---

## Tasks Performed
- Created a new Angular standalone application.
- Explored the generated project structure.
- Examined important configuration files.
- Built the project using Angular CLI.
- Generated reusable components using Angular CLI.
- Configured Angular Routing.
- Created a Header component.
- Created Home page.
- Created Course List page.
- Created Student Profile page.
- Verified navigation between pages.

---

## Angular CLI Commands Used
```bash
ng new student-course-portal --routing --style=css

ng generate component components/header

ng generate component pages/home

ng generate component pages/course-list

ng generate component pages/student-profile

ng serve

ng build
```
